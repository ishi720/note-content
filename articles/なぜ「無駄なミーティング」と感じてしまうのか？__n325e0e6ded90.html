なぜ「無駄なミーティング」と感じてしまうのか？

元記事を開く ／ 公開日: ／ いいね: 0


はじめに

皆さんの職場では、無駄なミーティングが行われていませんか？どんなミーティングが無駄だと思いますか？

この記事では、無駄なミーティングとは何か、そしてどう改善できるかを私なりにまとめました。

無駄なミーティングの正体

さっそくですが、なぜ無駄なミーティングだと感じるのか。それは、あなたがそのミーティングに不満を持っているからです。

しかし、あなたが不満に思っているミーティングを、他のメンバーは良いミーティングだと思っている可能性があります。逆に、あなたが良いと思っているミーティングに、他のメンバーが不満を感じている可能性もあります。

つまり、「無駄かどうか」は客観的な基準ではなく、参加者の主観的な満足度によって決まるのです。

満足度を上げるには？

全員が満足するミーティングは難しいかもしれません。しかし、不満に思う人を減らすことで、そのミーティングの満足度は上がるはずです。

では、どんなときに人は不満を感じるのでしょうか？以下は、多くの人が「無駄だ」と感じやすい共通パターンです。

よくある「無駄」のパターン

  • 目的が定まっていない → 何のために集まったのか分からない

  • 決定権のある人がいない → 話しても何も決まらない、打合せ後に覆される可能性あり

  • 人数だけ多い → 自分が必要な理由が分からない

  • チャットで十分な内容 → 時間を拘束された感覚になる

  • 話がまとまらない → 時間だけが過ぎていく

  • 結論が最初から決まっている → 意見を求められても意味がない

  • ほぼ雑談で終わる → 仕事が進まない

  • 定例的に集まるだけ → 惰性で続いている感覚になる

ちなみに私が特に不満に思うのは、「次のミーティングに参加してもらっていい？」と言われて参加したら、冒頭で「それでは、○○システムの概要について説明をお願いできますか？」と突然振られるパターンです。話を振るなら事前に言っておいてほしいし、説明できなかったらどうするつもりなのかと、不満に感じます。

ミーティングの種類別・押さえておきたいポイント

ミーティングは大きく3種類に分けられます。それぞれの特徴と、主催者・参加者それぞれが気をつけるべきポイントを整理しました。

① 何かを決定するミーティング

未確定事項について、有識者を集めて決定する場です。

主催者がやるべきこと

  • 決定事項・判断材料・決定権者を明確にする

  • 決定権者が不在だと後から覆る可能性があるので注意

参加者がやるべきこと

  • 事前資料に目を通し、自分の意見を整理する

  • 空中戦にならないよう、具体的に発言する

② 情報を共有・報告するミーティング

進捗管理や情報共有のために実施します。

主催者／共有する側がやるべきこと

  • 伝えたい内容を整理し、想定質問を用意する

  • 一方的な報告で終わらせない

参加者／共有される側がやるべきこと

  • 事前に共有された情報を確認しておく

  • 漏れや問題点がないかを確認する

③ 学習・教育を目的とするミーティング

メンバーの知識の底上げのために実施します。

主催者／教える側がやるべきこと

  • 参加者のレベルに合わせた内容を用意する

  • 自己満足にならないよう、理解度を確認する

参加者／教わる側がやるべきこと

  • 事前に疑問点を洗い出しておく

  • 分からないことは遠慮なく質問する

今日からできる改善アクション

無駄なミーティングを減らすために、主催者・参加者が個々で意識する必要があります。

  • アジェンダを事前に共有する

  • 資料を事前に用意する、事前に目を通す

  • 参加者を必要最小限にする

  • 決定権者の出席を確保する

  • 終了時に決定事項と次のアクションを確認する

  • 「このミーティング、本当に必要？」と自問する

  • 自分の役割が不明なら事前に確認する

  • 論点がズレたら指摘する

忘れてはいけない「第三者の目線」

実はもう一つ、大切な視点があります。それは、ミーティングに参加していない第三者の目線のあなたです。

ミーティングが雑談だけで終わっても、参加したメンバーは満足しているかもしれません。しかし第三者のあなたから見たら、「そのミーティング、意味あるの？」「時間の無駄じゃない？」と思いますよね。

無駄だと判定されてしまうと、ルールが追加されてどんどん規制されていきます。参加者の満足度だけでなく、第三者から見ても意味のあるミーティングだったかを意識することが大切です。

おわりに

この記事についてまとめます。

  • 無駄なミーティングと感じるのは、あなたがそのミーティングに不満を持っているから

  • 不満を感じる人を減らすことで、ミーティングの満足度は上がる

  • ミーティングの種類（決定・共有・教育）に応じた準備と心構えが大切

  • 主催者も参加者も、事前準備と目的意識を持つ

  • 参加者だけでなく、第三者の目線でも振り返る

より良いミーティングライフを送るために、まずは一つでも改善アクションを試してみてください。
